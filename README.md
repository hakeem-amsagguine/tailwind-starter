## ðŸš€ Getting Started

Follow these instructions to set up and run the project locally.

### 1. Clone the Repository

```bash

```

Make sure you have [Node.js](https://nodejs.org/) installed. Then run:


Generate the `dist/styles.css` file by running:
